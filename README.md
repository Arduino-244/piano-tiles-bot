# Arduino Piano Tiles Bot
 Automatically plays Piano Tiles using Photoresistors and Micro-Servos.

# Components used:
 - **4x** Micro-Servos
 - **4x** Photoresistors
 - **4x** Popsicle sticks
 - **4x** Piece of tinfoil (Ball-shaped)
 - Cables
 - Piece of cardboard or any other material that will not damage the screen.

# Libraries used:
 - **Servo**:
  Micro Servo library
